# Reference catalog

This directory will contain trusted metadata for independently maintained reference implementation repositories.

The catalog stores immutable source coordinates, trust status, compatibility, validation evidence, primary use cases, and supporting architectural choices. It does not contain or aggregate the complete workspaces.

Registration makes an example discoverable. Validation, vetting, and recommendation remain distinct states.
